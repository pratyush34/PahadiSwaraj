# PahadiSwaraj Backend Engine

This is the AI-assisted full-stack web development backend engine built using Node.js, Express, and Mongoose. It features automated e-commerce copy generation, structured history logging, full CRUD capability across 6 endpoints, query-based search filters, and strict global exception handling.

---

## How to run backend locally

Follow these step-by-step instructions to get your local development environment up and running.

### Prerequisites
Make sure you have the following installed on your machine:
* [Node.js](https://nodejs.org/) (v18.x or higher recommended)
* [npm](https://www.npmjs.com/) (comes pre-bundled with Node.js)
* [MongoDB](https://www.mongodb.com/try/download/community) (Local community server running on port `27017` OR an active MongoDB Atlas connection string)
